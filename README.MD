## 简介

[Battle city](https://github.com/hec9527/BattleCity)是经典的红白机游戏之一，其操作简单，玩法经典，同时易于上手，本项目包含游戏的多种版本，有非常完善的开发文档以及开发日志，并且经过严格的代码测试

项目适合新手前端玩家入门练习，也可以适合摸\*....🐶
